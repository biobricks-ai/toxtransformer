What is ToxTransformer? A decoder-only transformer that predicts 6,647 binary toxicity properties simultaneously from molecular structure (SELFIES). A single forward pass produces calibrated probabilities for all properties. Optional context — known property values earlier in the sequence — further improves predictions.

Why does logistic regression beat the decision tree? Trees discretize continuous probabilities into binary splits, losing calibration information — especially harmful with 1-7 features and small N. LogReg preserves the probability signal, learns optimal token weights, and can invert relationships via negative coefficients.

How Zero-Shot Transfer Works

1

Pretrained Model

ToxTransformer learns 6,647 toxicity properties jointly via a decoder-only transformer on SELFIES molecular sequences.

2

Semantic Matching

Claude AI searches the 6,647-token catalog and selects 1-7 relevant pretrained tokens for each new endpoint based on construct validity, positive rate alignment, and assay type.

3

Transfer & Adapt

Zero-shot: average matched tokens (no training). Adapter: fit a 1-7 feature logistic regression to learn optimal weights and handle inversions. Both use only ToxTransformer's pretrained representations.

Known Limitations & Failure Modes

CYP Substrate Confusion — The catalog lacks pure CYP substrate tokens. Available tokens conflate substrates with inhibitors. 3 CYP substrate endpoints were excluded from evaluation.

SIDER Breadth — Clinical organ toxicity endpoints (cardio, pulmonary, renal, dermatological) rely on single SIDER tokens that capture broad MedDRA adverse event classes, often 2-3x higher positive rates than the curated UniTox labels.

Class Imbalance — Dermatological toxicity endpoints have 85-92% positive rates, making AUC unreliable with very few true negatives.
